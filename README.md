# 🎮 TIC-TAC-TOE Game 

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![JavaFX](https://img.shields.io/badge/JavaFX-FF0000?style=for-the-badge&logo=java&logoColor=white)

A modern, desktop-based Tic-Tac-Toe game built with Java and JavaFX. This project features a clean graphical user interface and an **unbeatable AI opponent** powered by the Minimax decision-making algorithm.

## ✨ Key Features

* **Interactive GUI:** Built with JavaFX and JFoenix material design components for a smooth user experience.
* **Unbeatable AI:** The computer opponent uses the Minimax algorithm, making it mathematically impossible to lose against. It will always win or force a tie.
* **Real-time Game State:** Instantly evaluates win, loss, and tie conditions after every move.
* **Clean Architecture:** Follows strong Object-Oriented Programming (OOP) principles, neatly separating UI controllers from core game logic.
* **Instant Replay:** Easily reset the board and start a new game with a single click.

## 🛠️ Technologies Used

* **Language:** Java
* **UI Framework:** JavaFX
* **UI Components:** JFoenix
* **Core Algorithm:** Minimax (Game Theory)
